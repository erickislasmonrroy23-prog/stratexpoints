# StratexPoints — Strategic Execution Platform

> Hospital Punta Médica · *"Ciencia al Cuidado del Paciente"*

Plataforma SaaS de gestión estratégica integral impulsada por IA.

## Módulos
- 🏠 Dashboard Ejecutivo
- 🗺️ Mapa Estratégico BSC (4 perspectivas · 17 objetivos)
- 🔷 Hoshin Kanri X-Matrix (correlaciones interactivas)
- 🎯 OKR Tracker (Dirección Administración y Finanzas)
- 🎳 Bowling Chart KPIs (12 meses · semáforo)
- 🤖 Lector de Documentos IA (PDF, Excel, PPT, Word, CSV, imágenes)
- 📤 Exportación Excel + PDF

## Stack
React 18 · Vite 5 · Claude API (claude-sonnet-4-20250514)

## Instalación local
```bash
npm install
npm run dev
```
